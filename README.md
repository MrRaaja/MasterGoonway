⭐ 1️⃣ How to RESET / RESTART the bot
If you’re using systemd (autostart):

Restart:
sudo systemctl restart goonbot


Stop:
sudo systemctl stop goonbot


Start:
sudo systemctl start goonbot

Check status:
sudo systemctl status goonbot


This is the clean and safe way.
